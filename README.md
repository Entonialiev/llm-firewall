# LLM Firewall 🔥

**Межсетевой экран для больших языковых моделей (LLM/LLM Firewall)**  
*Стандарт A-CODE, версия 1.0*  
*Автор: Элшан Алиев (Lead Architect, A-CODE Standards)*

## Назначение

LLM Firewall — это middleware для защиты LLM от:
- **Промпт-инъекций** (prompt injections)
- **Джейлбрейков** (jailbreaks)
- **Утечек персональных данных** (PII)
- **Попыток системного оверрайда**

Протокол реализует принципы автоматизированного комплаенса (AC-2026) на уровне входящих запросов к ИИ-моделям.
